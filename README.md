# TaskFlow

A simple To-Do List web application made using HTML, CSS, and JavaScript.

## Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Dark and light themes
- Local storage support
- Live date and time

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Enter a task
2. Click on "Add Task"
3. Mark tasks as completed
4. Delete completed tasks
